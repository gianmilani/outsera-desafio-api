# worst-producers-movie-api

    Api responsável por obter o produtor com maior intervalo entre dois prêmios consecutivos,
    e o que obteve dois prêmios mais rápido.

## Conteúdo

### Stack

| Tecnologia        | Versão |
| ---               | -------|
| Java              | 17     |
| SpringBoot        | 3.4.2  |
| H2                | 2.3.232|

### Executando a Aplicação
    
    ./gradlew bootRun

### Executando Testes

    ./gradlew test
